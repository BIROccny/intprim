Detailed Description

The LinearSystem class provides general methods which are required by the KF linear filter.

This includes the generation of transition models, process noise models, and measurement models.

Gets the measurement model for spatial filtering.

This is the Jacobian associated with the partial differential of \( \boldsymbol{H}_t = \frac{\partial h(\boldsymbol{s}_t)}{\partial s_t} \). However, the partial derivative of a linear combination of basis weights and basis functions is simply the basis functions.

Returns
Matrix of dimension B x D containing the measurement model.
